# dam import

Выполняется следующая последовательность:
- Собирает список приложений уже установленных в системе
- Собирает список, какие будут пропущены при импорте
- Собирает список, какие добавятся новые
- Собирает, какие приложения будут удалены из системы
- Задает вопрос, вы хотите применить изменения в системе
- Подготовка системы. идет по списку добавляемых новых приложений из файла экспорта, 
смотрит есть ли в кэше докера уже загруженное приложение, если нет загружает.
- Удаляет приложения по списку
- Устанавливает приложения по списку

Внимание! Удаляются только приложения, кэш докера не чистится
